At this point it is important to note the fact that in AutoCAD software there are also interface components that are by default floating and docked panels or can be made floating and docked according to the user’s preference. These components include Properties panel, layer properties manager, etc. You can right click on any object you are drawing (creating) or editing and open the properties panel which you can implement as floating (not attached to fixed components of the interface) or docked (attached to fixed interface components) interface component. This topic will be continue sly mentioned during your exercise activities.

To further discuss about the matter over:
with respect to what comes below the level of the tabs, it is important to understand that below tabs are ribbons under them are panels containing specialized group of commands or tools that are placed horizontally arranged in a ribbon or every tab separated by an engraved line. The general level of hierarchy in accessing the commands or tools of AutoCAD are : Tabs + ribbons + panels + commands or tools (on recent interfaces). The older versions use the panel and tools bar system. Please take note of the fact that you can mouse hover on any tools of the panels and access general information about each tool. Short key command and further information on how and what to use the commands for. Right clicking on each of the tools in a panel you will find an options such as (a) Show tabs and (b) show panels and (c). These options will allow you to hide or un hide any of the tabs and panels in your interface as per your interest. You can also add any of the panels in to the list of the quick access bar discussed above in the interface figures. How ever if you would right click in the ribbon with out clicking on the tools you will only find the first two options. Right clicking in the tab areas will yield different dropdown list from what you will get in the ribbon. Please also take note of the fact that you can get a docked or float windows when you for example right click on a drawing element in your model space such a properties manager panel. Or if you would click on the layer tool from its panel instead , you will end up with the layer properties manager floating panel. Block editor windows are also floating windows. Understand that most commands must come together for the user to create a proper project.

Only drawing lines and other objects without using the OSNAP modes ,ORTHOMODE/ POLAR mode applications from the control panel would not yield a drawing that has precision. Similarly if a user applies the above modes and does not apply layers properly, would not create a project that is accessible for further modification (actually layers should be the first thing users should think about before starting to draw). Think that you need to coordinate among for example the draw panel (line, polyline, circle ,arc, polygon, etc. ), the modify panel (move, rotate, offset, etc.), the annotate panel (text, dimension, leader, table, etc.), the control panel (ORTHO, POLAR, OSNAP, etc.) and the layer panel (layer properties, on/off, freeze, etc. ) should come together for your successful drawing activity.

Therefore, in AutoCAD simply (minimally) put , you organize (layer panel), you set up for precision (control panel), you create (draw panel, annotate panel, and the like), you modify (modify panel), you prepare for plot and presentation ( layout tab) , you save (application icon/button or ctrl+s).

G. The Layer properties manager in 'model space' and in 'layout space'

Status, Name, Freeze, on/off, lock, Plot, Color, Linetype, Lineweight, Transparency, New VP Freeze, Description, Filters, etc.

There are five additional layer properties in the layout space that are specific to viewports and are not included while you are in the model space.
(to access the layer properties service: (1) type Layer in the command line, (2) home tab + layers panel + layer properties))

The standard AutoCAD software comes with only one or two default layers and the user needs to carefully create needed layers ad create standards for projects.

The use of layers guarantees future modification and management of your project. Unless you have carefully created layers from the start, you will be faced with a lot of time cleaning up your document which is not a best practice in AutoCAD.


Click on this image to further take a look at interface information of the layer properties manager

Comparison of layer properties in model space and layout space:

The layer properties in the model space are as indicated: 12 properties: 1:Status, 2: name, 3: freeze, 4: on/off, 5: lock, 6: plot, 7: color, 8: linetype, 9: lineweight, 10: transparency, 11: new viewport freeze and 12: descriptions. The layer properties in the layout space are as: 17 properties with additional 5 on top of what we have in model space: the additional 5 layer properties are: 1: Viewport freeze, 2: viewport color, 3: viewport line type, 4: viewport line weight and, 5: viewport transparency. The above five layer properties are used to add or modify properties of a layer in a specific view port with out affecting other viewports or the project in model space.

I. Plotting (The plot window)

    link: Interface A  Interface B  Interface C   Interface D   Interface E   Interface F   Interface G   Interface H   Interface J
You can publish and plot your project from both the model space and the layout space. To make use of the plot service of the software one needs to understand the difference between the model space and the layout space, since the latter is dedicated for organizing your project for plotting while the former is dedicated to creating and modifying. The following image shows some of the basic differences between the model space and the layout space. The layout space comes with ‘PLOT’ tab that allows to draw viewports of different shapes and paper sizes.
